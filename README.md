# Doctxing · 数海微庐

[![license](./image/license_github.svg)](https://github.com/Doctxing/blog/blob/master/LICENSE)

余名 Doctxing。或称 Doctxing Skywalker，亦有以 Xeno 呼之者。名虽数易，不过一符号耳，然符号之下，所寄者志也。

世人初见 `Doctxing` 之名，多误以为 “doctor”。然此非余本意。昔者立名之初，不过自觉好奇成性，见物则欲穷其理，遇事辄思究其源。书卷、文档、典籍、代码，皆如林木丛生，故取西洋文 `doc` 为意——*与其言医人之术，不若言求知之途*。

余生也晚，幸逢万象竞发之世。天机入算，星网贯空，山海可越于一屏之间。于虚网之中辟一隅小地，是为此博客。无华饰，无成制，不过手搓数页，以记平日所思所学。或有所得，则记之；或有疑惑，则书之；或一念偶起，亦不忍使其散于风中。

文字虽浅，志在求真。世事浩繁，学海无涯。愿以寥寥数行，记行旅之迹；以片言只字，存思辨之火。后来之人，或偶过此地，得其一隅启悟，亦足矣。若夫同好之士，偶驻其步，共论是非，商榷得失，则斯圃不独为余之所栖，亦可为群贤暂憩之所也。

是为序。

## 杂

此站构建，皆手作而成，未循成模板。以 `pug` 为骨，以 `Tailwind CSS v3` 为饰。虽 `Tailwind v4` 已行于世，然念及兼容之宜，故仍守 v3 之制。

盖 v4 所适者，乃 Safari 16.4 以上、Chrome 111 以上、Firefox 128 以上之浏览器。若欲顾及旧制之器，则 v3.4 尚为稳妥之选。

> Tailwind CSS v4.0 is designed for Safari 16.4+, Chrome 111+, and Firefox 128+. If you need to support older browsers, stick with v3.4 until your browser support requirements change.

然观其新版本，诸般细节多有精进，制度亦愈臻完备。依理言之，今之环境迁移亦未必为难，唯为稳妥起见，暂从旧章而已。
